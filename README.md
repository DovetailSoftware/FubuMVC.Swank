FubuMVC Swank
=============

[Check out this sample](http://www.mikeobrien.net/FubuMVC.Swank)

This needs to be set for the asset pipeline to work:

```xml
<configuration>
  <system.webServer>
    <modules runAllManagedModulesForAllRequests="true" />
  </system.webServer>
<configuration>  
```
